# Binary Trees in C

This project implements a set of functions to handle *binary trees* in C, including creation, deletion, traversal, and node property calculations.

---

## Project Files

│
├─ 0-binary_tree_node.c        # Create a binary tree node
├─ 1-binary_tree_insert_left.c # Insert a node to the left
├─ 2-binary_tree_insert_right.c# Insert a node to the right
├─ 3-binary_tree_delete.c      # Delete entire tree
├─ 4-binary_tree_is_leaf.c     # Check if a node is a leaf
├─ 5-binary_tree_is_root.c     # Check if a node is a root
├─ 6-binary_tree_preorder.c    # Pre-order traversal
├─ 7-binary_tree_inorder.c     # In-order traversal
├─ 8-binary_tree_postorder.c   # Post-order traversal
├─ 9-binary_tree_height.c      # Measure tree height
├─ 10-binary_tree_depth.c      # Measure node depth
├─ 11-binary_tree_size.c       # Count tree size
├─ 12-binary_tree_leaves.c     # Count leaves
├─ 13-binary_tree_nodes.c      # Count nodes with children
├─ 14-binary_tree_balance.c    # Compute balance factor
├─ 15-binary_tree_is_full.c    # Check if tree is full
├─ 16-binary_tree_is_perfect.c # Check if tree is perfect
├─ 17-binary_tree_sibling.c    # Find sibling of a node
├─ 18-binary_tree_uncle.c      # Find uncle of a node
├─ binary_tree_print.c         # Print the tree
├─ binary_trees.h              # Struct definition and function prototypes
└─ main.c / <n>-main.c         # Test files for each task

---
